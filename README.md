#### Libraries

This project uses:
- [React.js][React]
- [Redux][Redux]
- [Pusher][Pusher] for implementing websocket
- [Emojione-picker][emojione-picker] for implementing emoji
- [Mailboxer][mailboxer] for messaging feature
- [Devise][devise] for authentication feature
